List of checks run by the modelchecker on the schematisation
------------------------------------------------------------

These are the checks run by the modelchecker on the schematisation.
The checks are listed in the order in which they are run.
The beta checks (error code 1300) are optional, and can be disabled for testing
by running threedi-modelchecker from the command line with the `--allow-beta` flag.
